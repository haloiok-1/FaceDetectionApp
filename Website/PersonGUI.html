PersonGUI Documentation

Documentation of the PersonGUI Class

Class: PersonGUI

General Function:

The PersonGUI class provides a graphical user interface for managing a list of known persons. It allows users to view the list of persons, open their photo folders, delete persons, and take photos of them using the integrated Phototaker class.

Functions:

  • __init__(master, working_directory): Initializes the PersonGUI object with the master window and the working directory.
  • start(): Starts the PersonGUI application.
  • get_persons(): Retrieves the list of known persons from the persons.json file.
  • create_grid_for_persons(): Creates a grid layout to display information and options for each person.
  • open_fileexplorer_for_person(person): Opens the file explorer to view the photo folder of a specific person.
  • delete_person(person): Deletes a person from the list and their associated photo folder.
  • open_photo_taker(person): Opens the photo taker for a specific person.
  • update_grid(): Updates the grid layout with the latest information.
