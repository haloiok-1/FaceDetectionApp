Documentation of the Trainer Class

Class: Trainer

General Function:

The Trainer class is responsible for training a face recognition model using images from a specified directory. It detects faces in images, assigns labels to them, and trains the model to recognize those faces. The class provides methods to load images, check data integrity, train the model, and save labels.

How Haar Cascades Work:

Haar cascades are machine learning-based classifiers used for object detection. They work by applying a series of feature-based classifiers to subsections of an image, known as "Haar-like features." These features are simple patterns, such as edges, corners, or lines, that are computationally efficient to compute.

During the training phase, the Haar cascades algorithm learns to identify patterns that are characteristic of the object being detected, such as faces. These patterns are stored as a cascade of classifiers, with each classifier making a decision based on a subset of features. During detection, the cascade is applied to different regions of the image, and if a region matches the pattern described by the cascade, it is considered a positive detection.

Haar cascades are particularly effective for tasks like face detection because they can quickly scan an image at multiple scales and orientations, making them suitable for real-time applications.

Functions:

  • __init__(working_dir, image_dir, cascade_path): Initializes the Trainer object with the specified working directory, image directory, and cascade path.
  • load_images(): Loads images from the specified image directory, detects faces, and assigns labels.
  • check_data(): Checks if the training data is valid and sufficient for training.
  • train(): Trains the face recognition model using the loaded images and labels.
  • save_labels(): Saves the labels assigned to the training data.
  • process(): Performs the complete training process, including loading images, training the model, and saving labels.
  • import_labels_from_json(json_file): Imports labels from a JSON file.
  • getTrainingStatus(): Returns the current training status.
  • getAmountOfPhotos(): Returns the total number of photos used for training.
