Documentation of the FaceDetector Class

Class: FaceDetector

General Function:

The FaceDetector class is responsible for real-time face detection using OpenCV and Tkinter in Python. It captures video from the camera, detects faces, and performs facial recognition. The class provides a user-friendly interface for detecting and displaying recognized faces in real-time.

Functions:

  • __init__(parent, working_directory): Initializes the FaceDetector object with the specified parent widget and working directory.
  • detect_faces(): Performs face detection on the captured video stream using OpenCV and Haar cascades.
  • display_camera_stream(): Displays the camera stream with detected faces in a Tkinter label widget.
  • start(): Starts the FaceDetector application, initiating the Tkinter main loop.
  • stop(): Stops the FaceDetector application and releases the camera resources.
