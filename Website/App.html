Person Detector Application

Documentation of the App Class

Class: App

General Function:

The App class serves as the backbone of the Python GUI application. It encapsulates the core functionality and serves as the entry point for user interactions. By managing various components and modules, it provides a seamless experience for users to capture, train, and detect faces. The class ensures data integrity, user feedback, and system stability throughout the application lifecycle.

Functions:

  • import_csv_to_list(file_path): A function that reads a CSV file and returns its content as a list of strings.
  • check_if_camera_is_connected(): Checks if a camera is connected by attempting to capture a camera image.
  • submit(): A method called when the user submits the form to capture a person. It validates the entered information and then saves it to a JSON file.
  • change_directory(): A method called when the user wants to change the working directory. Opens a file selection dialog to choose a new working directory.
  • saveInJSON(): A method that saves the entered person data to a JSON file.
  • start_phototaker(): Starts the photo-taking mode to capture photos of the person.
  • start_face_training(): Starts the face training mode to generate and train facial recognition data.
  • worker_face_training(face_recognizer): A helper method for face training that runs in the background and performs the training.
  • start_facedetector(): Starts the face detection mode to recognize faces in real-time.
  • showPersons(): Starts the GUI to display the captured persons.
  • fn_amount_of_photos(): A method that counts the number of existing photos in the working directory.
