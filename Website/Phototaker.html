Documentation of the Phototaker Class

Class: Phototaker

General Function:

The Phototaker class provides functionality to capture photos using the computer's camera. It allows users to take photos, set profile pictures, and save them to a specified directory. The class displays the camera stream in a tkinter window and enables users to interact with the camera through buttons.

Functions:

  • __init__(parent, person, directory_path): Initializes the Phototaker object with the parent window, person information, and directory path for storing photos.
  • start(): Starts the photo-taking process and displays the camera stream.
  • display_camera_stream(): Displays the camera stream in the tkinter window and detects faces in real-time.
  • take_photos(): Initiates the process of taking photos.
  • worker_take_photos(): Worker function to capture photos in a separate thread.
  • save_photos(): Saves the captured photos to the specified directory.
  • current_amount_of_photos(): Returns the current number of photos in the photo directory.
  • set_profile_pic(): Sets the captured photo as the profile picture for the person and updates the JSON file accordingly.
