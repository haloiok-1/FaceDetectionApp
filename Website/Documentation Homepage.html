Overall Information

Dependencies

This application requires the following Python libraries:

    Internal Libraries

  • os
  • json
  • csv

    • External Libraries

  • OpenCV
  • Tkinter
  • Pillow

Features

The application offers a comprehensive range of functionalities designed to streamline your experience:

  • Verify camera connectivity with a quick check, ensuring seamless interaction with camera-related features.
  • Efficiently submit and store comprehensive person information, including name, last name, age, and gender, for robust data management.
  • Customize the working directory for photo storage, providing flexibility and organization tailored to your preferences.
  • Initiate face training procedures to optimize facial recognition accuracy, enabling precise identification in various scenarios.
  • Activate face detection functionalities to effortlessly identify and track faces within images or live camera feeds, enhancing security and usability.

Installation

To run the application, follow these steps:

  1. Ensure Python is installed on your system.
  2. Install the required libraries using pip:

    3. ! If you are able to run Shell scripts navigate to the folder and run installation.sh !

    pip install opencv-python opencv-contrib-python
    pip install pillow
    pip install tk
  3. Download or clone the repository.
    4. FaceDetectionApp on GitHub
    git clone https://github.com/haloiok-1/FaceDetectionApp
  4. Navigate to the application's directory and run the script:
    5. python main.py
