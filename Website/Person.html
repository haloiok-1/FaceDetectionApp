Documentation of the Person Class

Class: Person

General Function:

The Person class represents an individual with attributes such as first name, last name, age, gender, profile picture path, and photo folder path. It provides methods to verify profile picture and photo folder paths, display person information, and retrieve the current amount of photos in the photo folder.

Attributes:

  • firstname: The first name of the person.
  • lastname: The last name of the person.
  • age: The age of the person.
  • gender: The gender of the person.
  • profile_pic_path: The file path to the profile picture of the person.
  • photo_folder_path: The directory path where the person's photos are stored.

    • Methods:

  • verify(): Verifies the existence of the profile picture and photo folder.
  • display_info(): Displays information about the person.
  • current_amount_of_photos(): Returns the current number of photos in the photo folder.
